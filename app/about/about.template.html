This Angular Application is primary about architecture. It is based on two Pluralsight Courses and their correspondant Git Hub Repos.

Goals

  • Get better at Angular
  • Keep Angular projects orginized and modular
  • Use 'controller as' syntax
  • Use ui-router
  • Get resolve to work
  • Use gulp for the usal suspects like compiling LESS/SASS, bundeling and minification
  • But also use gulp for autoprefix css, ng-annotate and strip console.log
  • Get the same LESS architecture with Bootstrap as with SASS and Foundation
  • Repeat Bootstrap a little bit

Resources

  • Angular Patterns Clean Code
  • Angular Application Development
  • John Papas repo with sample projects and a styleguide for Angular
  • Lukas Ruebbelkes repo with sample projects
